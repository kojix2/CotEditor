Change settings in CotEditor on Mac

Use CotEditor settings to change options for the appearance and behavior of the editor and for how to handle documents.

In the CotEditor app on your Mac, choose CotEditor > Settings, then click a tab:

  • General: Specify how to handle documents, save, and more.
  • Appearance: Set the appearance of documents, such as font, line height, and colors.
  • Window: Customize the default window appearance such as window size or line number visibility.
  • Edit: Specify how the editor behaves while editing.
  • Mode: Vary the editor’s behaviors depending on the document’s syntax.
  • Format: Set the default format for documents such as line ending type and text encoding, and manage syntaxes.
  • Snippets: Define preset text to insert with your preferred keyboard shortcuts or by dropping files onto the editor.
  • Key Bindings: Customize keyboard shortcuts to execute menu commands.
  • Donation: Support the CotEditor project by offering coffee.

    • To learn about the options in a pane, click at the bottom-right corner of the settings window.
