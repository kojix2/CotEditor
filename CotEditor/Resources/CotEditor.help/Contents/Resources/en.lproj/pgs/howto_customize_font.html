You can temporary change your documentâ€™s font.

  1. Choose Format > Font > Show Fonts.

Select the Reset to Default command in the Font menu to reset the font in the active window to your default setting.

These changes apply only to the window currently active. To change the default font for all new documents, choose CotEditor > Settings > Appearance.
